# Spotify Markup Clone

A collaborative frontend project created for an exam.

## Pages
- `index.html`
- `playlist.html`
- `album.html`
- `artist_home.html`
- `about_artist.html`
- `grid_albums.html`
- `list_albums.html`
- `song.html`

## Tech
- HTML5
- SCSS (Sass partials)
- CSS (compiled)

## Project Structure
- `sass/abstracts/` variables, mixins, fonts
- `sass/layout/` layout partials (header, sidebar, footer, main layout)
- `sass/pages/` page-specific styles
- `assets/` images and UI assets
